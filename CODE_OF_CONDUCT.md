# Code of Conduct

Этот проект следует
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Кратко

- Уважайте других участников, независимо от опыта, бэкграунда, уровня сертификации или языка.
- Конструктивная критика — да. Личные нападки, харассмент, дискриминация — нет.
- Дайвинг-контекст: не насмехайтесь над новичками за «глупые» вопросы про снаряжение.
  Безопасность подготовки начинается с возможности спросить.
- Поддерживайте позиционирование проекта: ассистент подготовки, не авторитет безопасности.
  Не подталкивайте других нарушать собственное обучение или сертификацию.

## Куда жаловаться

Открытые issue с тегом `coc` или приватно через GitHub Security advisories
(см. [`SECURITY.md`](SECURITY.md)).

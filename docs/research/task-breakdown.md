# Explicitly Out of Scope

Do NOT build yet:

- accounts;
- backend;
- cloud sync;
- social features;
- decompression features;
- weather APIs;
- AI assistant;
- marketplace.

---

# MVP Success Criteria

The MVP succeeds if users:

- reopen it before dives;
- customize checklists;
- report fewer forgotten items;
- say preparation feels calmer and more structured.

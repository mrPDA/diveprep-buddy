# DivePrep Buddy — Feature Specification

## MVP Features

### Feature 1 — Dive Context Selection

Users select:

- shore/boat dive;
- cold/warm water;
- night dive;
- travel dive.

---

### Feature 2 — Contextual Checklist

Generate dive-preparation checklist.

---

### Feature 3 — Buddy Check Flow

Structured buddy verification.

---

### Feature 4 — Gear Reminder System

Remind users about commonly forgotten gear.

---

### Feature 5 — Travel Preparation

Travel-specific reminders and prep steps.

---

# Non-MVP

- dive logging;
- decompression calculations;
- social feed;
- instructor marketplace.

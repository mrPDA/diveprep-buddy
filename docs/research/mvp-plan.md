# DivePrep Buddy — MVP Plan

## MVP Goal

Reduce forgotten preparation steps and pre-dive stress.

---

# Core MVP

```text
Dive context
→ contextual checklist
→ buddy verification
→ readiness confirmation
```

---

# Included

- dive type selection;
- checklist generation;
- buddy-check flow;
- travel prep;
- gear reminders.

---

# Excluded

- dive logging;
- decompression planning;
- advanced dive computer sync;
- social platform features.

---

# MVP Success Signal

Users repeatedly use the app before dives.
